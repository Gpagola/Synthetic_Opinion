Deja aquí los archivos de tipografía (.woff2 preferido).

Nombres esperados por el CSS (@font-face en src/styles.css):

  personae-sans-400.woff2   -> texto normal (weight 400)
  personae-sans-500.woff2   -> semibold / botones (weight 500)
  personae-sans-600.woff2   -> titulares (weight 600)
  personae-mono-400.woff2   -> datos / etiquetas monoespaciadas (weight 400)

Si tus archivos tienen otros nombres, díselo a Claude o renómbralos a estos.
También puedes añadir .woff como segundo formato para navegadores antiguos.
